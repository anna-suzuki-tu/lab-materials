# lab-materials

DeSFE Lab（東北大学 鈴木杏奈研究室）の教材・配布資料の公開ホスティング用リポジトリ。
GitHub Pages で配信し、Notion・授業スライド・QRコードから直接リンクする。

## 収録教材

| 教材 | パス | 再生ページ |

## 追加のしかた

1. `教材名/` フォルダを作ってファイルを置く（ファイル名は半角英数字・ハイフン）
2. commit & push → 数分で `https://anna-suzuki-tu.github.io/lab-materials/<パス>` で公開される
