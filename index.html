DESFE LAB — EDUCATION PORTAL

DeSFE Lab 教育ポータル

東北大学 流体科学研究所・自然構造デザイン研究分野（鈴木杏奈研究室）
探究の術を、研究を通して磨き上げる。 — DeSFE Lab（鈴木研）の教育方針

鈴木研は、探究する人が集う場所です。ここは「研究をする場所」である以上に、研究を通して探究の術（すべ）を磨き上げる場所——だから私たちは、研究成果そのものよりも、研究を通じてどう成長するかを重視します。

このサイトは、DeSFE Lab で実際に使っている教育資料を公開するポータルです。研究室メンバーの学習のためであると同時に、私たちが「探究をどう教え、どう設計するか」を研究室の外の方にも伝えるために公開しています。学生・教育関係者の方は、授業やゼミで自由にお使いください。

💎 Lab Values

鈴木研の4つの価値
すべての活動の判断基準。このサイトの教材はすべて、この4つの実装です。
VALUE 01
自己管理 / Self-management
研究の主導権は自分にある。時間・計画・体調を自分で設計する。Research Log と Quarter Plan は自己管理の道具。管理されるのを待たない。
VALUE 02
誠実さ / Integrity
データにも、人にも、自分にも正直であること。うまくいかない結果や停滞も隠さず共有する。「詰まって沈黙」は誠実さの反対。
VALUE 03
敬意 / Respect
人・アイデア・時間への敬意。議論では Work（研究）を批判し、Peer（人）を攻撃しない。相手の時間を尊重した準備をして臨む。
VALUE 04
共創的探究 / Collective Inquiry
研究は個人戦ではない。Paper Card・Inquiry Cycle という共通言語で、互いの探究を加速させる。文化は全員でつくる。
価値は貼り紙ではなく、日常の行動で示すもの——具体的な実装は Lab OS の6章で。
